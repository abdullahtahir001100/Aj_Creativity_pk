## Technical Stack  

- React (Vite) — Fast dev & bundling  
- React Router v6 — Dynamic routing  
- Express.js (Node.js) — REST API  
- MongoDB — Flexible NoSQL DB  
- Tailwind CSS + SCSS — UI system  
- AOS — Scroll animations  
- Leaflet — Interactive maps  
- Linux + Vercel — Deployment  

---

## Features & Security  

- Interactive maps with Leaflet (store locations, tracking)  
- Scroll-based animations using AOS  
- HSTS enabled (secure connections)  
- Dynamic + protected routing  
- Scalable product schema (MongoDB)  
- Fully responsive UI
